# homebrew-formulae

Access point for installing my public [Homebrew](https://brew.sh/) formulae.

## Tapping

To make the formalae available to `brew`, tap this repo:

```shell
brew tap olets/formulae
```

## Installing formulae

Install formulae with

```shell
brew install <formula name>
```
